# 《Rust编程之道》随书源码示例

### 使用说明：

```shell
$ git clone https://github.com/ZhangHanDong/tao-of-rust-codes.git
$ cd tao-of-rust-codes
$ cargo doc
$ open target/doc/tao_of_rust/index.html
```
